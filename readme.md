# Estemia Territorio

Proyecto web desarrollado durante las prácticas de SMR como parte del rebranding y renovación de la antigua plataforma **Territorio-Master**.

La página ha sido diseñada con el objetivo de modernizar la imagen del proyecto, mejorar la organización visual y ofrecer una estructura más clara, limpia y accesible para la gestión y visualización de información territorial.

---

## 🌐 Características principales

- Diseño web moderno y organizado.
- Navegación sencilla e intuitiva.
- Estructura modular y escalable.
- Organización clara del contenido territorial.
- Rebranding completo de Territorio-Master hacia Estemia.
- Código estructurado y preparado para futuras mejoras.
- Separación eficiente de recursos y componentes.

## ⚙️ Instalación

Clonar el repositorio:

```bash
git clone https://github.com/TheSalvaOne/Estemia-territorio.git
```

Acceder al proyecto:

```bash
cd Estemia-territorio
```

Abrir `index.html` en el navegador.
